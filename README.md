# Hardwire

Landing page for Hardwire — the deployment layer / harness for physical, real-world AI.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design source

Built from the Figma moodboard mockup (`public/reference/mockup.png`). Fonts: Motorblock (wordmark), Geist Pixel (card titles), Geist Mono (card body / tagline).
